# simple-excel

```
Excel读写操作简化版
文件的读取和写入只支持xlsx格式的Excel
```


## Excel写入

- 导出功能根据“PHP_XLSXWriter”项目改写
- PHP_XLSXWriter项目地址(https://github.com/mk-j/PHP_XLSXWriter)
- 简单使用示见 test 目录


## Excel读取

- 导出功能根据“PHP_XLSXWriter”项目改写
- PHP_XLSXWriter项目地址(https://github.com/nuovo/spreadsheet-reader.git)
- 简单使用示见 test 目录
